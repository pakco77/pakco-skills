# Genmitsu Web CAM MVP — 产品架构决策

> 2026-05-09 Socratic 对话产出，PMO + 用户确认。后续迭代以此为基础。

## 核心架构：Wizard + 画布混合

```
首页二选一
├── 🎓 我是新手 → Wizard 4步 → 首次成功 → 解锁画布
└── ⚡ 直接开始 → 画布工作台（胶囊工具栏+项目标签+2D/3D分屏+浮动面板）
```

## Wizard 4 步详解

| 步 | 名称 | 画布 | 用户操作 |
|---|------|------|---------|
| 1 | 机器确认 | ❌ | 静态确认清单（机器/刀具/材料已就绪） |
| 2 | 选项目 | ❌ | 示例项目（预计算刀路）或 导入自己文件 |
| 3 | 画布微调 | ✅ 首次介入 | 拖拽移动/缩放设计、双击编辑文字、切换材料、展开高级调参 |
| 4 | 导出完成 | ✅ 保留 | G-code 导出 + 「试试自己的设计」引导 |

## 首次成功策略

1. 示例项目（Genmitsu 标牌）预计算刀路 → 3 分钟出实物
2. 成功后引导「试试你自己的设计」→ 导入文件再跑一次
3. 两次成功后解锁完整画布工作台

## 画布工作台

- **胶囊工具栏**：吸附画布上方，三段（设计工具｜刀路策略｜快捷操作）
- **项目标签栏**：顶部切换，一个项目=一个独立画布
- **2D/3D 分屏**：MVP 3D 为静态等轴测占位
- **浮动属性面板**：选中对象时在旁边弹出，非固定右侧栏

## Step 3 用户可调参数

| 层级 | 参数 | 交互 |
|------|------|------|
| 🟢 可见即可改 | 材料类型 | 下拉切换，自动重算全部推荐参数 |
| 🟢 可见即可改 | 设计位置/大小 | 画布上拖拽+缩放手柄 |
| 🟢 可见即可改 | 文字内容 | 双击文字编辑（示例项目） |
| 🟡 折叠区 | 主轴转速/进给率/切深 | 滑块，高级调参折叠默认关闭 |

## 范围约束

| 维度 | 决策 |
|------|------|
| MVP 模式 | 仅 CNC，激光架构预留 |
| 3D 预览 | 静态等轴测占位，不做 WebGL |
| 首版机型 | 硬编码 3018-PROVer V2 |
| 账户 | 不涉及登录，纯本地工具 |
| 离线 | 纯离线，不依赖网络 |

## 竞品差异化

- **示例项目预计算刀路**：Easel 和 easyCrave 都没有 → 核心差异化
- **Wizard+画布混合**：easyCrave 是纯 Wizard（无可交互画布），Easel 是纯画布（无引导）
- **Genmitsu 硬件预配置**：参数针对本机优化，无需用户手动配置机器参数
