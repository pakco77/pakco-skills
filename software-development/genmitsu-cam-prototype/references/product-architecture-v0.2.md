# Genmitsu CAM MVP v0.2 — 产品架构设计

> 产出日期：2026-05-09 | 原型：`2_B_迭代冻结产出/原型.html`

## 首页（Home）

- **布局**：单栏居中，最大宽 960px
- **新建项目**：虚线 accent 色卡片，醒目 CTA → 进入 Wizard
- **历史项目**：4 列网格，每卡片含缩略图+名称+材料+时间
- **社区作品**：3 列网格，含作者名

## 新手路径（Wizard）

| # | 步骤 | 核心交互 |
|---|------|---------|
| 1 | 机器确认 | 静态清单（3 项），非实时检测 |
| 2 | 选项目 | 示例项目卡片 + 导入文件 |
| 3 | 画布微调 | 拖拽/缩放/编辑文字/切换材料/高级调参/生成刀路 |
| 4 | 导出完成 | 导出 G-code + 引导进画布/试自己设计 |

- **画布介入节点**：Step 3（生成刀路后出现「继续」按钮→Step 4）
- **首次成功**：示例项目预计算刀路，3 分钟出实物

## 画布工作台

- **项目标签栏**：顶部 tab 式，可切换/关闭/新建（最少 1 个）
- **胶囊工具栏**：吸附画布上方，三段式（设计工具 | 刀路策略 | 操作），圆角全 pill
- **2D/3D 分屏**：可拖拽分隔条调整比例（200-600px），3D 可关闭/再打开
- **3D 预览**：CSS 3D transform 立方体线框占位，非 WebGL
- **底部状态栏**：机型 | 材料 | 刀路状态

## 关键设计决策

| 决策点 | 结论 |
|--------|------|
| 默认材料 | 软木（非 PCB） |
| Wizard 画布 | 画布在 Step 3 介入，而非 Step 1 |
| 3D 深度 | MVP 静态占位，不做 WebGL |
| CNC/激光 | 仅 CNC，架构留激光扩展口 |
| 机型 | 首版硬编码 3018-PROVer V2 |
| 示例项目 | Genmitsu 文字标牌（V 雕） |

## 与 v0.1 的区别

v0.1 是三栏固定布局（工具|画布|参数），v0.2 改用胶囊工具栏最大化画布空间，2D+3D 分屏，项目标签化。首页从二选一卡片改为完整 Home 页。
