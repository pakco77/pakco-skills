# 重启写作Agent

在新对话中说任意一句：
- 「重启写作Agent」
- 「加载 pakco-writing-system」
- 「公众号写作」

Agent 会自动：
1. 扫描 Obsidian 写作目录
2. 检查最新周报数据
3. 加载文风参考
4. 汇报当前状态

## 快速指令

| 你想做的 | 说 |
|---------|-----|
| 写新文章 | 「用我的风格写一篇关于XXX的」 |
| 深度特稿 | 「用卡兹克风格写XXX」 |
| 生成周报 | 「跑周报」或「生成周报」 |
| 诊断现状 | 「分析我的公众号差在哪」 |
| 改进文章 | 「这篇按10万+标准打磨」 |
| 提取样式排版 | 「提取这篇的样式」「用XX的风格排HTML」 |
| 生成HTML | 「生成HTML预览」 |
| 启动排版工具 | 「启动 wechat-typesetting」 |
