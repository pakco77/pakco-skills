# 文章发布前审查清单

## 必须项（缺一不发）

- [ ] 固定开场：「Hi 朋友们，我是Pakco。」或变体
- [ ] 固定结尾：「我是Pakco。一个不会写代码的产品经理，在学AI。」
- [ ] 关注引导：明确一句「点个关注」+ 理由
- [ ] 下一篇预告：具体标题或方向（不要"敬请期待"）
- [ ] 所有【待填】占位符已删除或已填
- [ ] 错别字检查（特别注意：没想白→没想到、coffee店→咖啡店）
- [ ] "Offer"统一小写"offer"

## 建议项

- [ ] 标题写5版，找3人投票
- [ ] 开头100字内有钩子（悬念/反差/场景）
- [ ] 至少3处"你"（共鸣插入）
- [ ] 至少2句可截图转发的金句
- [ ] 结尾有一个让读者沉默的问题（不是答案）
- [ ] 字数控制在3000-5000（公众号最佳阅读区间）
- [ ] 无超过150字的段落（手机屏会累）
- [ ] 无「家人们」「宝子们」等直播语气
- [ ] 无「在当今这个日新月异的时代」等AI味套话

## 叙事段检查

- 每个叙事段 ≤ 80 字
- 超过则拆成 2-3 小段
- 最后 3-5 句可用短排句制造节奏
- 对话分行独立成段

## 标题对比检查

| 你的标题 | 10万+版特征 |
|---------|-----------|
| 结论先行型 | → 改成过程先行（"我做了X"→"我以为A，结果B"） |
| 陈述句型 | → 改成悬念/冲突/数字型 |
| 工具题材 | → 改成人物题材（加"我""他""她"） |
